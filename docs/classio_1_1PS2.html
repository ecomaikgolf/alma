Detailed Description

PS2 Keyboards pushes a hex keycode each time a key is pressed then another when it's released, for example:

0x1E : 'a' pressed 0x9E : 'A' released

There is no '/' por upper/lower keys, PS2 only comprehends the keys available in the keyboard, upper/lower keys do not exist on it... right?

We have to generate them by playing with pressed/released combinations
